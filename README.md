### 列表
- 定义列表
> alpha = ['a', 'b', 'c', 'd', 'e', 'f', 'g']
- 访问列表中的元素
```
>>> alpha[0]
'a'
>>> alpha[3]
'd'
>>> alpha[-1]
'g'
```
- 切片
```
>>> alpha[0:4]  #列表元素的下标从0开始，第一个元素的下标为0
['a', 'b', 'c', 'd'] #从下标0开始取，下标为0的元素可以取到，为4的元素取不到
>>> alpha[3:-1] #取下标3至-1的值，不包括-1
['d', 'e', 'f']
>>> alpha[3:] #如果想取到最后一个
['d', 'e', 'f', 'g']
>>> names[::2] #每两个元素取一个
['a', 'c', 'e', 'g'] 
```
